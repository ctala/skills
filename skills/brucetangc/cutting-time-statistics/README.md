# 切割时间统计 Skill

## ⚠️ 使用前准备

**需要艾威时间模板文件才能使用！**

模板文件路径：`/home/admin/.openclaw/media/inbound/艾威时间模板.xlsx`

模板文件不需要上传，用户需要自行准备。

## 📋 功能

从激光切割 PDF 文件中提取**切割时间**，按**板厚汇总**，生成**完整艾威模板格式**的 Excel 报表。

## 🚀 使用方法

```bash
python3 skills/激光切割时间统计/calculator.py <压缩包路径> [输出文件]
```

### 示例

```bash
# 基础用法（自动生成英文文件名）
python3 skills/激光切割时间统计/calculator.py AW-25-2072.7z
# 输出：AW-25-2072_Cutting_time_statistics.xlsx

# 指定输出文件
python3 skills/激光切割时间统计/calculator.py 050.7z 自定义文件名.xlsx
```

## 📊 处理逻辑

### 1. 厚度提取
从 PDF 文件名提取厚度：
- `1.5mm.pdf` → 1.5mm
- `3Mn.pdf` → 3mm
- `10mm.pdf` → 10mm
- `nm8mm.pdf` → 8mm（耐磨板）

### 2. 切割时间提取
只提取 **第一页 Job data** 中的 Cutting time：
- 格式 1：`AWGY-2026-050-2m10m332 2_5.0_O2.PAR 8.44 kg 5.04min 60.8 % 12`
- 格式 2：`10332 2_5.0_O2.PAR 6.86 kg 2.80min 25.7 % 15`
- 每个 PDF 只提取一个时间值

### 3. 汇总逻辑
- 相同板厚的时间**合并到同一行**（如 5mm + 5Mn 合并到 5mm 行）
- **D 列留空**（不填单个 PDF 时间）
- **E 列直接填总和**（保留两位小数）
- **删除无数据的厚度行**
- **自动更新公式行号**（F/H/L 列）
- **合计放在备注行**（保留两位小数，居中，带边框）

## 📁 输出格式（完整艾威模板）

| 列 | 字段 | 说明 | 公式 | 格式 |
|----|------|------|------|------|
| A | 生产日期 | 留空（用户填写） | | |
| B | 厚度 mm | 1, 1.5, 2, 3, 4... | | |
| C | （空） | | | |
| D | 理论时间分钟 | **留空** | | |
| E | 理论时间总计分钟 | 同厚度合计 | 直接填数值 | 0.00 |
| F | 结算时间分钟 | 加 10% 时间补偿 | `=E 行号+(E 行号*F$18)` | |
| G | 加工费 1 分钟 | 6.33 元（固定） | | |
| H | 加工费总价 | 结算时间×单价 | `=G 行号*F 行号` | |
| I | 材料费 | 用户填写 | | |
| J | 打孔攻丝总价 | 用户填写 | | |
| K | 折弯总价 | 用户填写 | | |
| L | 总计 | 所有费用合计 | `=H 行号+I 行号+J 行号+K 行号` | |

### 备注行（最后一行）

| A 列 | L 列 |
|------|------|
| 备注： | `=SUM(L19:L 最后数据行)` |
| 左对齐，边框 | 居中，两位小数，边框 |

### 10% 时间补偿

**F18 = 0.1** 表示**10% 的时间补偿率**，用于补偿：
- 工件定位
- 设备调试
- 板材更换
- 空移时间
- 其他辅助时间

**计算公式：**
```
结算时间 = 理论时间 × 1.1
加工费 = 结算时间 × 6.33 元/分钟
总费用 = 加工费 + 材料费 + 打孔攻丝费 + 折弯费
```

## 🔍 自检功能

技能内置自检功能，处理完成后自动验证：
- ✅ F 列公式：`=E 行号+(E 行号*F$18)`
- ✅ H 列公式：`=G 行号*F 行号`
- ✅ L 列公式：`=H 行号+I 行号+J 行号+K 行号`
- ✅ 合计公式：`=SUM(L19:L 最后数据行)`

如果公式错误会报错提示，确保输出文件正确。
