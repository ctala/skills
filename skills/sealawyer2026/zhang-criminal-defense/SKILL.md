# 刑事辩护全流程AI助手 🏛️

> 专业刑事辩护律师的AI助手 | 6大阶段 | 21个模板 | 全流程覆盖

## 🎯 适用场景

- 刑事案件接案洽谈
- 侦查阶段辩护工作
- 审查起诉阶段辩护
- 审判阶段出庭辩护
- 上诉申诉代理
- 刑罚执行阶段

## 📋 阶段一：接案洽谈

### 1.1 案件初步评估
```
你是一位资深刑事辩护律师。请根据以下信息对案件进行初步评估：

案件类型：[盗窃/诈骗/故意伤害/其他]
案件阶段：[侦查/审查起诉/审判]
当事人身份：[嫌疑人/被告人/家属]
涉案金额/情节：[具体信息]

请提供：
1. 案件难度评估（1-5星）
2. 可能的辩护方向
3. 风险点提示
4. 收费建议区间
5. 下一步工作计划
```

### 1.2 委托合同起草
```
请根据以下信息起草刑事辩护委托合同：

委托人：[姓名/单位]
当事人：[姓名]
涉嫌罪名：[罪名]
案件阶段：[阶段]
代理权限：[一般授权/特别授权]

合同应包含：
1. 委托事项
2. 律师权利义务
3. 委托人权利义务
4. 费用条款
5. 保密条款
6. 风险提示
```

## 📋 阶段二：侦查阶段

### 2.1 会见笔录整理
```
请整理以下会见笔录，提取关键信息：

[粘贴会见笔录]

输出格式：
1. 当事人陈述要点（时间线）
2. 有利情节梳理
3. 不利情节分析
4. 需要核实的问题
5. 下一步会见重点
```

### 2.2 取保候审申请
```
请起草取保候审申请书：

当事人：[姓名]
涉嫌罪名：[罪名]
羁押地点：[看守所]
申请理由：[具体理由]

申请书应包含：
1. 申请人信息
2. 申请事项
3. 事实与理由（法律依据）
4. 保证方式
5. 保证人信息/保证金金额
```

### 2.3 不予批捕法律意见
```
请撰写不予批准逮捕法律意见书：

案件：[简要案情]
当事人：[姓名]
提请批捕机关：[公安机关]

意见书应包含：
1. 基本案情
2. 不予批捕的法律依据
3. 具体理由（证据不足/无社会危险性/情节轻微等）
4. 类似案例参考
5. 结论请求
```

## 📋 阶段三：审查起诉阶段

### 3.1 阅卷笔录制作
```
请根据以下卷宗材料制作阅卷笔录：

[粘贴卷宗摘要]

输出要求：
1. 证据清单（分类整理）
2. 证据分析（证明力评估）
3. 证据矛盾点
4. 缺失证据提示
5. 辩护要点提炼
```

### 3.2 质证意见起草
```
请针对以下证据起草质证意见：

证据类型：[物证/书证/证人证言/鉴定意见等]
证据内容：[简要描述]
质证重点：[合法性/真实性/关联性]

质证意见应包含：
1. 对证据合法性的意见
2. 对证据真实性的意见
3. 对证据关联性的意见
4. 对证明力的评价
5. 质证结论
```

### 3.3 不起诉法律意见
```
请撰写不起诉法律意见书：

案件：[简要案情]
当事人：[姓名]
审查起诉机关：[检察院]

意见书应包含：
1. 案件事实
2. 不起诉的法律依据（刑法/刑事诉讼法）
3. 具体理由（情节轻微/证据不足/等）
4. 类似不起诉案例
5. 结论请求
```

## 📋 阶段四：审判阶段

### 4.1 辩护词生成
```
请根据以下信息撰写辩护词：

案件：[简要案情]
被告人：[姓名]
涉嫌罪名：[罪名]
辩护思路：[无罪/罪轻/从轻减轻]

辩护词结构：
1. 首部（标题、称呼）
2. 辩护意见
   - 事实认定部分
   - 法律适用部分
   - 量刑情节部分
3. 结论
4. 尾部（律师签名、日期）
```

### 4.2 庭审发问提纲
```
请制作庭审发问提纲：

发问对象：[被告人/证人/鉴定人]
发问目的：[澄清事实/揭示矛盾/证明情节]
案件焦点：[争议焦点]

提纲格式：
1. 身份核实类问题
2. 案件事实类问题
3. 量刑情节类问题
4. 预设追问（根据回答）
```

### 4.3 量刑辩护意见
```
请撰写量刑辩护意见：

罪名：[罪名]
基准刑：[建议刑期]
从轻情节：[自首/立功/认罪认罚/等]
从重情节：[如有]

意见应包含：
1. 法定从轻减轻情节
2. 酌定从轻情节
3. 类案量刑参考
4. 具体量刑建议
5. 适用缓刑的可行性分析
```

## 📋 阶段五：上诉申诉

### 5.1 上诉状起草
```
请起草刑事上诉状：

原审法院：[法院名称]
案号：[案号]
上诉人：[姓名]
判决结果：[判决内容]
上诉理由：[具体理由]

上诉状应包含：
1. 当事人信息
2. 上诉请求
3. 上诉理由（事实认定错误/法律适用错误/程序违法/量刑不当）
4. 证据清单
5. 此致法院
```

### 5.2 申诉材料准备
```
请准备刑事申诉材料：

原审情况：[审级/结果]
申诉理由：[新证据/法律适用错误/等]

材料清单：
1. 申诉书
2. 原审裁判文书
3. 新证据材料
4. 法律依据
5. 类似改判案例
```

## 📋 阶段六：执行阶段

### 6.1 减刑假释申请
```
请起草减刑/假释申请材料：

服刑人员：[姓名]
服刑监狱：[监狱名称]
已服刑期：[时间]
改造表现：[简要描述]

材料应包含：
1. 申请书
2. 改造表现证明
3. 财产刑履行情况
4. 家庭情况说明
5. 法律依据
```
