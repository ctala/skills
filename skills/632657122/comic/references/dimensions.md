# Comic Dimensions Reference

## Art

| Value | Best For | Recommended Mapping |
| --- | --- | --- |
| `ligne-claire` | knowledge comics, clean explanation, European clear-line style | `flat-illustration` |
| `manga` | manga pacing, expressive emotion, strong panel energy | `manga` |
| `realistic` | realistic storytelling and historical subjects | `photoreal` |
| `ink-brush` | East Asian brushwork and ink storytelling | `ink-brush` |
| `chalk` | tutorial, classroom, or blackboard aesthetics | `chalk` |

## Tone

| Value | Best For |
| --- | --- |
| `neutral` | rational and educational explanation |
| `warm` | friendly and comforting stories |
| `dramatic` | strong contrast and dramatic tension |
| `romantic` | softer and more decorative emotion |
| `energetic` | bright, fast, lively pacing |
| `vintage` | historical and period atmosphere |
| `action` | impact, speed, and motion |

## Layout

| Value | Best For |
| --- | --- |
| `standard` | the most general 3- to 6-panel page |
| `cinematic` | large scenes and camera-driven composition |
| `dense` | more information on one page |
| `splash` | a page dominated by one big image |
| `mixed` | irregular panel structure |
| `webtoon` | vertical scrolling pages |

## Aspect

Recommended defaults:

- `3:4`
- `4:3`
- `16:9`

## Language

Follow the user's language by default.

If speech bubbles, captions, or title text are required, state the target language explicitly in the prompt.
