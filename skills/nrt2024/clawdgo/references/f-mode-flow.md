# F 模式对抗竞技场——每轮输出格式

每轮6步，连续5轮（已在启动 opt-in 时告知用户）。

## 步骤1 红方构造攻击

```
══ 红方攻击（第N轮）══
攻击维度：[S1/O2/E3等] | 手法：[名称]
[完整攻击内容]
红方意图：[目的]（蓝方视角不可见）
```

## 步骤2 视角切换

输出隔离提示：`--- 视角切换：蓝方登场 ---`（蓝方只能看攻击内容，不能看意图）

## 步骤3 蓝方盲判

**此时绝对不参考步骤1的红方意图。**

```
══ 蓝方判断（第N轮）══
威胁识别：[类型] | 危险信号：[1-5条] | 决定行动：[行动] | 信心：[高/中/低]
```

## 步骤4 裁判评分

```
══ 裁判评分（第N轮）══
红方：XX/50（真实性X | 复杂度X | 执行X）
蓝方：XX/50（识别X | 信号X | 决策X）
本轮胜者：[红方/蓝方] | 点评：[一句话]
```

评分标准：
- 红方满分50：真实性15 + 复杂度20 + 执行15
- 蓝方满分50：识别20 + 信号20 + 决策10

## 步骤5 复盘

红蓝双方复盘 + 关联安全知识点

## 步骤6 升级机制

- 红方连胜则下轮难度+1级
- 蓝方连胜3轮解锁高阶防御徽章

→ 进入下一轮

## Arena 称号体系（按蓝方防御得分）

铜壳卫士（蓝胜1-2轮）｜银爪斗士（3轮）｜金甲强龙（4轮）｜无敌龙神（全胜5轮）

## 约束

- 蓝方判断时绝对不参考红方意图
- 红方不得故意低质量放水
- 裁判不得自我放水
